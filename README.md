# Attendance Tracker

A minimal dashboard for tracking student attendance in real time.

## Features
- Real-time attendance tracking
- Student profiles
- Modern UI built with React and Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
